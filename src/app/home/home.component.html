About the APP Summary of this app

This is a simple Angular application setup with the latest Angular version of the library (v.17).

The app will stand as a boilerplate that will act as a starting point for other applications.

This setup includes basic functionalities like:

  • Routing & Navigation
  • Angular Material
  • SCSS language for stylesheets
  • Standalone architecture
 How to get started Concrete steps to have you up & running in no time

Prerequisites

  1. Make sure you have the NodeJS environment installed on your system.
  2. Install GIT on your system.
  3. Clone locally this repository.

Concrete steps

  1. open a terminal window to the location of the project
    • cd path/to/project
  2. Install the required dependencies:
    • npm run install
  3. Run the project
    • npm run start
  4. In the terminal you should see the URL and port where the application is being server
    • Usually this is http://localhost:4200

If everything went ok, now you are all setup to make changes to this project and see the updates live.
